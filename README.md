
# Bite Count

Bite Count adalah chatbot yang membantu menghitung kalori makanan langsung lewat chat. tanpa perlu instal aplikasi, cukup kirim pesan sebutkan makanan yang kamu ingin konsumsi, dan Bite Count akan memberikan informasi kalori serta rekomendasi olahraga.
